# 📊 Customer Behavior Analysis Project

## 📌 Overview
This project focuses on analyzing customer behavior to uncover meaningful insights that support business decision-making. It includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, and dashboard creation.

The objective is to understand purchasing patterns, customer preferences, and key factors influencing buying decisions.

---

## 📂 Dataset
- The dataset contains customer transaction and behavioral data  
- Key features include:
  - Customer ID  
  - Product details  
  - Purchase amount  
  - Ratings and reviews  
  - Location and other attributes  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)  
- SQL (PostgreSQL / MySQL / SQL Server)  
- Power BI (Dashboard creation)  
- Gamma (Presentation)  
- Jupyter Notebook  

---

## 🔄 Project Steps

### 1. Data Loading
- Imported dataset into Python using Pandas  
- Connected SQL database for querying  

### 2. Data Cleaning
- Handled missing values  
- Removed duplicates  
- Converted data types  
- Ensured data consistency  

### 3. Exploratory Data Analysis (EDA)
- Performed analysis using Pandas  
- Identified:
  - Frequently purchased products  
  - Customer spending patterns  
  - Product ratings distribution  

### 4. SQL Analysis
- Wrote queries to extract insights such as:
  - Top-performing products  
  - Average ratings  
  - Customer spending trends  
  - Sales distribution  

### 5. Dashboard Creation
- Built an interactive Power BI dashboard to visualize insights  
- Focused on:
  - Sales overview  
  - Customer behavior  
  - Product performance  

### 6. Reporting
- Summarized findings in a structured report  
- Created a presentation using Gamma  

---

## 📊 Dashboard
The Power BI dashboard includes:
- Sales and revenue overview  
- Customer segmentation insights  
- Product performance analysis  
- Interactive filters

## 📈 Results & Insights
- Identified top products based on ratings and sales  
- Analyzed customer spending behavior

- Generated insights for improving business strategies

This project demonstrates end-to-end data analytics skills including data cleaning, SQL querying, and dashboard creation. The insights derived can help businesses make data-driven decisions and improve customer experience.
